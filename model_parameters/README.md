## Model parameters
The final model parameters are stored in the 'UNet_medium_e150_bs16_lr0004.pth.tar' file. The name indicates U-Net-medium is the chosen model and was trained for 150 epochs, while the hyperparameters are learning_rate = 0.0004 and bach_size = 16. Additionally, The performance metrics for the training are stored in 'properties_UNet_medium_e150_bs16_lr0004.json' for training, validation and test data. A summary of the tuning process can be found in the 'properties_tuning_maxepoch300_grp10_red3_br2.txt' file. This includes validation and test scores for the best model and information about all 100 trials.
